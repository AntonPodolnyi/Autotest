1.Для запуска сценария необходимо скачать и установить NUnit 2.6.3
(доступен на: https://launchpad.net/nunitv2/trunk/2.6.3/+download/NUnit-2.6.3.msi)

2. В ходе выполнения сценарий попытается создать каталог С:\_TestReports (можно изменить в исходном коде проекта) для сохранения в нем отчета и скриншота. Соответственно, до запуска теста необходимо убедиться, что диск C:\ существует и к нему имеется полный доступ.

2. После установки и запуска NUnit выбрать: File -> Open Project... 
Выбрать файл проекта: Mercury.Autotest -> Bin -> Release -> Mercury.Autotest.dll

3. После загрузки проекта в NUnit и отображения его в дереве тестов нажать кнопку Run.

4. Отработав, сценарий напишет в поле "Text Output" информацию о завершении и сохранении отчета и скриншота с автоматически сгенерированными именами в директории С:\_TestReports. 


Примечание.
Сценарий тестировался на браузерах Chrome 51.0.2704.103 и Firefox 47.0.1 в системе Windows 7(x64), .NET Framework 4.5. 
Разработан на C# (MS Visual Studio 2013) с использованием WebDriver 2.53.1, NUnit Framework 3.4.1, RelevantCodes ExtentReports 1.4.9.
